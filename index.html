Chapter 1: History of the web.

Sir Tim Berners-Lee invented the World Wide Web in 1989. (Please click the link to learn more. It is very interesting and will provide a further understanding on what you will be learning.) If you decide not to click the link, basically what happens is:

  1. Sir Tim is working, and finds it a hassle to log on to each computer to view the information kept on it
  2. He decides to create HTML(HyperText Markup Language), URI(Uniform Resource Identifier) also called URL, and HTTP(Hypertext Transfer Protocol.) Also, he creates the first Web page editor/viewer(WorldWideWeb.app), and the first Web server ("httpd")
  3. This leads to the open source creation of the Web and yada-yada, so-on, the Web is here.
