Typography

Headings

You can add a normal heading with <h1>...<h6>.

EXAMPLE

h1 • Example Header

2.5em / 40px*

h2 • Example Header

2.00em / 32px*

h3 • Example Header

1.50em / 24px*

h4 • Example Header

1.25em / 20px*

h5 • Example Header

1.00em / 16px*

h6 • Example Header

0.875em / 14px*

*Conversions based on 16px browser default size

Code 
            
              <h1>h1 Example Header<p class="weak">2.5em/40px*</p></h1>
              ...
              <h6>h6 Example Header<p class="weak">0.875em/14px*</p></h6>

You can add a secondary header to any heading by using the <small> tag. Or, you can add the .weak class to any <p> tag. The text in the second option is a fixed size as seen in the example above.

Buttons

Custom ready made styled buttons for quick access.

Add btn rnd-btn or btn sqr-btn to the buttons class.

Rounded Buttons

EXAMPLE
Code 
              
              <button type="button" class="btn rnd-btn normal">Normal</button></a>
                ...
              <button type="button" class="btn rnd-btn error">Error</button></a>

Square Buttons

EXAMPLE
Code 
              
                <button type="button" class="btn sqr-btn normal">Normal</button></a>
                ...
                <button type="button" class="btn sqr-btn error">Error</button></a>
